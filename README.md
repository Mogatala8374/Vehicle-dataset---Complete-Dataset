# 🚗 Vehicle Price Prediction Using Machine Learning

## 📌 Project Overview

This project aims to predict the selling price of used cars using Machine Learning Regression algorithms. The project includes complete data preprocessing, exploratory data analysis (EDA), feature engineering, feature selection, model training, evaluation, and visualization through a dashboard.

The objective is to identify the key factors affecting vehicle prices and compare the performance of different regression models.

---

# 📂 Dataset

* **Dataset Name:** Car Details v3
* **Source:** CarDekho
* **Target Variable:** `selling_price`

The dataset contains information about used cars, including:

* Car Name
* Manufacturing Year
* Selling Price
* Kilometers Driven
* Fuel Type
* Seller Type
* Transmission
* Owner Type
* Mileage
* Engine Capacity
* Maximum Power
* Number of Seats

---

# 🎯 Project Objectives

* Perform data cleaning and preprocessing.

* Engineer new features to improve prediction accuracy.
